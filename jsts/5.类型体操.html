6.partial实现原理 #

  • [P in keyof T]
  • keyof 指的是把我们一个对象里面的 键值对里的键key给罗列取出来，并把它们联合起来形成一种联合类型
  • in在这里充当一个遍历的作用
  • 把keyof T进行一个个遍历并且每个都单独拿出来生成新的 "键值对"
typescript
// 内部就会变成这样
type Partial<T> = {
  [P in keyof T]?: T[P];
};

type newQunyou = Partial<QunYou>;

// ↓
type Partial<QunYou> = {
  [P in 'name' | 'age' | 'isLsp' | 'sex']?: QunYou[P];
};

// ↓
type Partial = {
  name?: QunYou["name"];
  age?: QunYou["age"];
  isLsp?: QunYou["isLsp"];
  sex?:QunYou["sex"];
};

// ↓
type Partial = {
  name?: string;
  age?: number;
  isLsp?: boolean;
  sex?: '男' | '女' | 0 | 1;
};

10.const断言 #

  • 表达式中的任何字面量类型都不应该被扩展；
  • 对象字面量的属性，将使用 readonly 修饰；
  • 数组字面量将变成 readonly 元组。
typescript
let x = "hello" as const;
type X = typeof x; // type X = "hello"

let y = [10, 20] as const;
type Y = typeof y; // type Y = readonly [10, 20]

let z = { text: "hello" } as const;
type Z = typeof z; // let z: { readonly text: "hello"; }

10.Exclude #

  • 从 Exclude 的定义来看，就是判断 T 是否继承于 U，如果是，则返回 never，否则返回 T。
  • type Exclude<T, U> = T extends U ? never : T;
typescript
interface IPerson {     // IPerson extend IMan  为true
  name: string,
  age: number,
  sex: 0 | 1,
}

interface IMan {
  name: string,
  age: number,
}

// IPerson 
type Man = Exclude<IMan, IPerson> // 等效于 type Man = never
